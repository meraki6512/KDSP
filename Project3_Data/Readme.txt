This dataset includes long-term (about 10 months) check-in data in New York city from Foursquare from 12 April 2012 to 16 February 2013.
It contains a file in tab separated format. The file contains 8 columns, which are:

1. User ID (anonymized)
2. Venue ID (Foursquare)
3. Venue category ID (Foursquare)
4. Venue category name (Fousquare)
5. Latitude
6. Longitude
7. Timezone offset in minutes (The offset in minutes between when this check-in occurred and the same time in UTC)
8. UTC time

The file dataset_NYC.txt contains 227428 check-ins in New York city.
